Création d'une Politique de mot de passe

Afin de garantir une bonne sécurité de notre infrastructure. Il est recommandé par l’ANSSI de respecter quelques règles dans la politique d’élaboration des mots de passe. Pour rappel un mot de passe fort : Est unique et non réutilisable pour une autre utilité ; Ne possède pas de lien avec la personne ; Est composé de 12 caractères MINIMUM ; Est composé de caractères Alphanumérique et de caractère spéciaux ; Est renouvelé régulièrement, 90 jours est un bon compromis.

Installation d'un MSI par GPO

Cette procédure à pour but d’installer par GPO, un programme avec l’extension .MSI, sur l’ensemble des postes d’un domaine, d’une OU. Cette procédure décriera les étapes pour installer Firefox ESR. Préparation Création d’un dossier Partagé Créer sur le serveur un dossier partagé. Ajouter les droits de lectures aux ordinateurs du domaine. Récupération de l’installateur Récupérer sur le site de l’éditeur, l’installateur au format MSI. copier l’installateur dans le dossier partagé.
