Création d'une Politique de mot de passe

  • Windows
  • ActiveDirectory
  • GPO
  • Password

Afin de garantir une bonne sécurité de notre infrastructure. Il est recommandé par l’ANSSI de respecter quelques règles dans la politique d’élaboration des mots de passe. Pour rappel un mot de passe fort  :

  • est unique et non réutilisable pour une autre utilité
  • ne possède pas de lien avec la personne
  • est composé de 12 caractères MINIMUM
  • est composé de caractères Alphanumérique et de caractère spéciaux
  • est renouvelé régulièrement, 90 jours est un bon compromis

Pour plus de détail visité l’ANSSi et plus particulièrement la recommandation

  • Ajouter les droits de lectures aux ordinateurs du domaine

Récupération de l’installateur

  • Récupérer sur le site de l’éditeur, l’installateur au format MSI
  • copier l’installateur dans le dossier partagé

Création de la GPO

Dans la fenêtre “Group Policy Management” créer une nouvelle GPO nommée “InstallPackage”

Faire ensuite un clic-droit sur la GPO puis sélectionner “Edit” dérouler l’arborescence:

|--Computer Configuration
  |--Policies
    |-- Software Settings
      |-- Software Installation

faire un clic droit sur “Software Installation”

clic droit
  > new
    > Package

Sélectionner l’installateur et choisir la méthode “Assigned”

Dans les filtres de sécurité, choisir: Utilisateur authentifié

Postes Clients

Sur chaque poste clients connectés à l’Active Directory il faut soit:

  1. attendre le redémarrage de la machine
  2. faire la commande: 
    gpudate /force

    puis redémarrer la machine
