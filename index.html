Procédures BTS

Installation de l'agent sur un poste Linux

Serveur Web Installation de l’agent Zabbix Récupération des sources Dans un terminal wget https://repo.zabbix.com/zabbix/5.4/debian/pool/main/z/zabbix-release/zabbix-release_5.4-1%2Bdebian10_all.deb Installation du paquet Dans un terminal sudo dpkg -i zabbix-release_5.4-1+debian10_all.deb modification des configurations de l’agent Ouvrir le fichier /etc/zabbix/zabbix.agentd.conf en tant que root sudo vim /etc/zabbix/zabbix.agentd.conf modifier les valeurs en ajoutant l’adresse IP du serveur Zabbix Server=SERVER_ZABBIX ServerActive=SERVER_ZABBIX Hostname=NOM_DE_LA_MACHINE recharger le service Agent sudo systemctl restart zabbix-agent.service

Ajout d'un agent dans le Dashboard

Serveur Zabbix Connexion au serveur Ouvrir dans un navigateur l’adresse du serveur Zabbix Ajout du nouvel Hote Dans Configuration/Hosts, cliquer sur “Create host” Puis ajouter les informations correspondantes au serveur et enregistrer Dashboard Dans Monitoring/Dashboard, cliquer sur “Edit dashboard” en haut a droite Ensuite, toujours en haut a droite dans le menu deroulant “ADD” choisir add page et choisir le nom adéquat au serveur (ex: Serveur Web) avec une periode de 30 seconde.
